# Haus

## wasm-pack

### Install

```sh
npm install
```

### Run in debug mode

```sh
# Builds the project and opens it in a new browser tab. Auto-reloads when the project changes.
npm start
```

### Build in release mode

```sh
# Builds the project and places it into the `dist` folder.
npm run build
```

### Run unit tests

```sh
# Runs tests
npm test
```

## Trunk

```sh
trunk serve
```
